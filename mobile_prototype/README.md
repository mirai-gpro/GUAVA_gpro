# グルメサポート AIコンシェルジュ - モバイルプロトタイプ

## アーキテクチャ

```
┌─────────────────────────────────────────────────────────────┐
│  Modal (一回限りのアセット生成)                               │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ GUAVA推論 → GS_canonical.ply + SMPLX binding data       ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
                              ↓ ダウンロード
┌─────────────────────────────────────────────────────────────┐
│  スマートフォン (完全オンデバイス実行)                        │
│  ┌─────────────────────────────────────────────────────────┐│
│  │ リップシンク入力 (音声→口形状パラメータ)                  ││
│  │        ↓                                                ││
│  │ SMPLX変形 (行列演算のみ - NNなし)                        ││
│  │        ↓                                                ││
│  │ Gaussian位置更新                                         ││
│  │        ↓                                                ││
│  │ Spark (WebGL2) 3DGSレンダリング                         ││
│  └─────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────┘
```

## 必要ファイル

```
mobile_prototype/
├── index.html          # メインアプリケーション
├── assets/
│   ├── GS_canonical_10k.ply    # 間引き済みGaussian (680KB)
│   ├── smplx_binding.json      # SMPLX頂点バインディング
│   └── lbs_weights.bin         # Linear Blend Skinning重み
└── README.md
```

## 使用技術

| コンポーネント | ライブラリ | 役割 |
|--------------|-----------|------|
| 3DGSレンダラー | [Spark](https://github.com/sparkjsdev/spark) | WebGL2ベースの高速3DGS描画 |
| 3Dエンジン | Three.js | シーン管理、カメラ制御 |
| 変形ロジック | 自作 (JavaScript) | SMPLX LBS (行列演算) |

## クオリティティア

| ティア | Gaussian数 | ファイルサイズ | 対象デバイス |
|-------|-----------|---------------|-------------|
| Low | 5,000 | ~340KB | 低スペック端末 |
| **Medium** | 10,000 | ~680KB | 一般的なスマホ |
| High | 50,000 | ~3.4MB | ハイエンド端末 |

## 実行方法

```bash
# ローカルサーバー起動
cd mobile_prototype
python -m http.server 8080

# ブラウザでアクセス
open http://localhost:8080
```
